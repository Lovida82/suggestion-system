새 Admin 비밀번호 재설정

⚠️ 문제 상황:
ADMIN 계정이 Firebase Auth에 이미 생성되어 있지만, 비밀번호를 모르는 상태입니다.

해결 방법 1: Firebase Auth에서 계정 삭제 (권장)

  1. Firebase Console 접속: https://console.firebase.google.com/
  2. 프로젝트 선택: suggestion-system-58def
  3. 왼쪽 메뉴에서 "Authentication" 클릭
  4. "Users" 탭에서 ADMIN@ajupharm.co.kr 계정 찾기
  5. 해당 계정의 메뉴(점 세개) 클릭하고 "Delete account" 선택
  6. 삭제 확인
  7. 다시 로그인 페이지에서 ADMIN / ajupharm으로 로그인 (자동 생성됨)

해결 방법 2: Firestore users 컬렉션 초기화

  1. Firestore Database 열고 users 컬렉션 선택
  2. ADMIN 계정과 연결된 문서 찾기 (employeeId가 ADMIN인 문서)
  3. 해당 문서에서 passwordChanged 필드를 false로 변경
  4. 또는 문서 전체 삭제

해결 방법 3: 직접 비밀번호 재설정 (고급)

Firebase Console의 Authentication 탭 Users에서 ADMIN 계정을 선택하고
"Reset password" 또는 우측 메뉴에서 비밀번호 직접 설정
