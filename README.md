## 시스템 개요

아주약품 제안제도 시스템은 직원들의 창의적인 아이디어를 체계적으로 관리하고 평가하는 시스템입니다.

### 주요 특징
- ✅ **3차 평가 시스템**: 1차 → 2차 → 3차 단계별 평가
- 💰 **유형효과 검증**: 금액 절감이 있는 제안은 검증담당자가 먼저 검증
- 🔍 **중복 제안 방지**: 유사 제안 자동 검색 기능
- 📊 **실시간 진행 상황**: 제안서 진행 단계를 실시간으로 확인
- 👥 **복수 권한 지원**: 한 사용자가 여러 역할 수행 가능

## 사용자 권한 체계

### 권한 종류 (복수 권한 가능)

| 권한 | 역할 | 사용 페이지 |
|------|------|------------|
| **일반사용자** (user) | 제안서 작성 및 조회 | dashboard.html |
| **1차평가자** (firstReviewer) | 팀/파트 단위 1차 평가 | leader.html |
| **2차평가자** (secondReviewer) | 부서 단위 2차 평가 | leader.html |
| **3차평가자** (thirdReviewer) | 최종 평가 권한 | leader.html |
| **유형효과검증담당자** (effectVerifier) | 유형효과 금액 검증 | verifier.html |
| **관리자** (admin) | 시스템 전체 관리 | admin.html |

### 로그인 시 페이지 이동 우선순위
1. **admin** 권한 → `admin.html` (최우선)
2. **effectVerifier** 권한 → `verifier.html`
3. **평가자** 권한 → `leader.html`
4. **user** 권한 → `dashboard.html`

## 평가 프로세스

### A. 무형효과만 있는 경우 (금액 절감 없음)
```
[제안 제출] → [1차 평가] → [2차 평가] → [3차 평가] → [최종 확정]
```

### B. 유형효과가 있는 경우 (금액 절감 발생)
```
[제안 제출] → [유형효과 검증] → [1차 평가] → [2차 평가] → [3차 평가] → [최종 확정]
```

### 프로세스 상세

#### 1단계: 제안 제출 (일반사용자)
- 제안 내용 작성
- **유형효과 선택**:
  - **무형효과만**: 바로 1차 평가로 진행
  - **유형효과 있음**:
    - 예상 절감 금액 입력 (원/년)
    - 산출 근거 500자 이상 필수
    - 증빙 자료 첨부 (선택)
    - 검증 완료 후 평가 진행

#### 2단계: 유형효과 검증 (검증담당자) - 유형효과 있는 경우만
- 제안 내용 및 산출 근거 검토
- **확정 금액 입력** (원/년)
- 금액 조정 사유 작성 (예상 금액과 다를 경우)
- 유형효과 점수 자동 계산
- **검증 완료** 후 1차 평가로 전달

#### 3단계: 1차 평가 (1차평가자)
- 팀/파트 단위 평가
- 평가 항목 입력:
  - 노력도 (5/10/15점)
  - 창의성 (5/10/15점)
  - 무형효과-품질 (0/5/7/10점)
  - 무형효과-안전 (0/5/7/10점)
  - 유형효과 (자동 표시, 수정 불가)
- 평가 의견 작성
- 중복 제안 검토
- **승인** / **반려** / **재검토 요청**

#### 4단계: 2차 평가 (2차평가자)
- 부서 단위 평가
- 1차 평가 점수 확인
- 동일한 평가 기준 적용
- 최종 승인 또는 반려

#### 5단계: 3차 평가 (3차평가자)
- 최종 평가
- 1차, 2차 평가 점수 확인
- 종합 평가 후 최종 확정
- 등급 및 보상 결정

## 페이지별 사용 가이드

### 1. index.html (로그인/회원가입)

#### 로그인 방법
1. **사번** 입력 (예: A197003)
2. **비밀번호** 입력
   - 초기 비밀번호: `ajupharm`
   - 첫 로그인 시 비밀번호 변경 필수

#### 계정 생성
- 관리자가 조직도를 업로드하면 자동으로 계정 생성
- 직원별로 초기 비밀번호는 `ajupharm`으로 통일

### 2. dashboard.html (일반사용자)

#### 주요 기능

##### 📊 대시보드
- 제출한 제안 수 확인
- 승인된 제안 수
- 검토 중인 제안
- 승인율 통계

##### ✏️ 제안서 작성

###### 기본 정보
- 제안번호: 자동 생성
- 접수일: 자동 입력
- 제안자/부서: 로그인 정보 기반 자동 입력
- 실시자/실시부서: 검색하여 선택
- 카테고리/우선순위: 드롭다운 선택

###### 제안 내용
- **제안내용**: 제목 (최소 5자)
- **기존실시내용**: 현황 및 정비이력 (최소 10자)
- **변경(신규)내용**: 개선 방안 (최소 10자)
- **투자비용**: 예상 비용 입력

###### 💰 유형효과 입력
1. **무형효과만 있음** 선택
   - 금액 절감 없음
   - 바로 평가 진행

2. **유형효과 있음** 선택
   - **예상 절감 금액** (원/년) 입력
   - **산출 근거** 500자 이상 상세 작성
     ```
     예시:
     - 기존: 월 100개 불량 × 개당 10,000원 = 월 1,000,000원 손실
     - 개선: 월 10개 불량으로 감소
     - 절감액: (100-10) × 10,000 × 12개월 = 10,800,000원/년
     ```
   - **증빙 자료** 첨부 (선택, 최대 3개)
   - ⚠️ **검증 완료 후 평가 진행**

###### 자체 평가 (단수 참여자)
- 노력도: 0~10점
- 창의성: 0~20점
- 유무형효과: 0~70점
- **총점 및 등급 자동 계산**

###### 기여도 평가 (다수 참여자)
- 참여자 추가 버튼 클릭
- 이름, 역할, 기여도(%), 부서 입력
- **기여도 합계 100% 필수**

###### 첨부파일
- 최대 5개 파일
- 개별 10MB 이내
- 지원 형식: PDF, DOC, XLS, PPT, HWP, JPG, PNG 등

##### 📋 내 제안서
- 제출한 제안서 목록
- 진행 현황 실시간 확인
  - 검증 상태 표시
  - 평가 단계 표시 (1차/2차/3차)
  - 현재 평가자 표시
- 상세보기 가능

##### 🔔 알림
- 평가 결과 알림
- 반려 사유 확인
- 수정 요청 사항

### 3. verifier.html (유형효과 검증담당자)

#### 주요 기능

##### 📊 대시보드
- **검증 대기**: 즉시 검증 필요 건수
- **오늘 검증**: 오늘 검증 완료 건수
- **월간 검증**: 이번 달 검증 총 건수
- **평균 조정률**: 확정금액/예상금액 비율

##### ⏰ 검증 대기
- 유형효과가 있는 제안서 목록
- 제안자, 제목, 예상 절감액 확인
- 대기일수 표시 (3일 초과 시 경고)

##### 💰 검증 작업

###### 제안서 정보 확인
- 제안 내용 전체 확인
- 예상 절감 금액 및 산출 근거 검토
- 첨부된 증빙 자료 다운로드

###### 검증 정보 입력
1. **확정 금액 (원/년)** 입력
   - 제안자가 입력한 예상 금액 검토
   - 실제 검증된 금액 입력
   - 유형효과 점수 자동 계산 및 표시

2. **금액 조정 사유** (예상 금액과 10% 이상 차이 시 필수)
   ```
   예시: "제안자는 연 1,200만원을 예상했으나,
   실제 원자재 가격을 고려하면 연 800만원이
   적정하다고 판단됨"
   ```

3. **검증 의견** 입력 (선택)

###### 검증 완료 옵션
- ✅ **검증 완료**: 확정 금액 승인, 평가 단계로 전달
- 📄 **추가 자료 요청**: 제안자에게 추가 정보 요청
- ❌ **검증 불가**: 제안서 반려

##### ✅ 검증 완료
- 검증 완료된 제안서 이력
- 예상 vs 확정 금액 비교
- 조정률 표시 (80% 이상: 녹색, 미만: 빨간색)

##### 📈 통계
- 총 검증 건수
- 평균 확정 금액
- 평균 조정률
- 평균 처리 시간
- 부서별 예상 금액 정확도

### 4. leader.html (평가자 전용)

#### 권한별 대시보드

평가자의 권한에 따라 자동으로 표시 내용이 변경됩니다:

| 권한 | 표시 내용 |
|------|-----------|
| 1차평가자만 | 1차 평가 대기 목록만 |
| 2차평가자만 | 2차 평가 대기 목록만 |
| 3차평가자만 | 3차 평가 대기 목록만 |
| 복수 권한자 | 모든 해당 평가 목록 (탭으로 구분) |

#### 주요 기능

##### 📊 대시보드
- **결재 대기**: 내가 평가해야 할 제안서 수
- **이번 주 처리**: 최근 7일간 처리한 건수
- **이번 달 처리**: 이번 달 총 처리 건수
- **평균 처리 시간**: 제출~승인 평균 소요 시간

##### ⏰ 결재 대기
- 내가 평가해야 할 제안서 목록
- 정보 표시:
  - 제안번호, 제안자, 제안부서
  - 이전 평가 점수 (2차, 3차 평가 시)
  - **검증 상태** 표시
    - ⏳ 검증 대기: 평가 불가
    - ✅ 검증 완료: 평가 가능
  - 중복 의심 마크 (자동 표시)
- **"평가하기"** 버튼 (검증 완료 건만 활성화)

##### 📝 평가 화면

###### 제안서 정보
- 제안 내용 전체 표시
- 기존실시내용 및 변경(신규)내용
- 자체평가 점수 확인

###### 중복 제안 검토
- **"중복 제안 검토"** 버튼 클릭
- 유사 제안 목록 모달 표시
  - 유사도 % 표시
  - 이전 제안의 평가 결과 확인
- 중복 판정 시 반려 사유 입력

###### 평가 입력 폼

1. **노력도** (드롭다운 선택)
   - 5점: 8시간 이내 (간단한 개선)
   - 10점: 80시간 이내 (1~2주 소요)
   - 15점: 80시간 이상 (장기 프로젝트)

2. **창의성** (드롭다운 선택)
   - 5점: 팀 내 유사 사례 있음
   - 10점: 본부 내 유사 사례 있음
   - 15점: 전사 최초 사례

3. **무형효과-품질** (드롭다운 선택)
   - 0점: 해당사항 없음
   - 5점: 품질 수준 소폭 향상
   - 7점: 경미한 불량 예방
   - 10점: 중대 불량 예방

4. **무형효과-안전** (드롭다운 선택)
   - 0점: 해당사항 없음
   - 5점: 작업 환경 개선
   - 7점: 경미한 사고 예방
   - 10점: 중대 사고 예방

5. **유형효과** (자동 표시, 수정 불가)
   - 검증담당자가 확정한 점수 자동 반영
   - 검증 완료 금액 표시

6. **종합점수** 및 **예상 등급** (자동 계산)

7. **평가 의견** (텍스트 입력)

###### 평가 항목별 평가 기준 툴팁
각 항목에 마우스를 올리면 상세 평가 기준 표시

###### 평가 결정
- ✅ **승인**: 다음 평가 단계로 전달
- ❌ **반려**: 제안서 반려 (사유 필수)
- 🔄 **재검토 요청**: 제안자에게 수정 요청

##### 📋 검토 중
- 내가 이미 승인한 제안서
- 다음 평가자 검토 대기 중
- 나의 평가 점수 및 의견 확인

##### ✅ 처리 완료
- 내가 처리한 제안서 목록
- 최종 승인/반려 상태
- 처리일, 평가 점수, 평가 의견 조회

##### 📊 통계
- 총 처리 건수
- 승인 건수 / 반려 건수
- 승인율
- 월별 처리 현황

### 5. admin.html (관리자 전용)

#### 주요 기능

##### 👥 사용자 관리 (강화)

###### 권한 관리
사용자 목록에 **권한 체크박스** 컬럼 추가:

| 체크박스 | 권한 |
|----------|------|
| ☑️ 일반사용자 | 기본 권한 |
| ☑️ 1차평가자 | 팀/파트 평가 |
| ☑️ 2차평가자 | 부서 평가 |
| ☑️ 3차평가자 | 최종 평가 |
| ☑️ 유형효과검증담당자 | 금액 검증 |
| ☑️ 관리자 | 시스템 관리 |

- **복수 선택 가능**: 한 사용자가 여러 권한 보유 가능
- **권한 일괄 변경**: 여러 사용자 선택 후 권한 일괄 부여
- **권한별 필터링**: 특정 권한자만 조회
- **권한 변경 이력**: 누가 언제 권한을 변경했는지 기록

###### 사용 예시
```
예: 김철수 팀장
☑️ 일반사용자 (제안 작성 가능)
☑️ 1차평가자 (팀원 제안 평가)
☑️ 2차평가자 (부서 제안 평가)
```

##### 📊 전체 현황 대시보드
- 전체 제안 진행 상황 (단계별)
- **병목 구간 표시** (처리 지연 건 강조)
- 평가자별 처리 현황
- 검증담당자 처리 현황

##### 👥 조직도 관리
- Excel 파일 업로드
- 템플릿 다운로드
- 조직도 현황 확인
  - 전체 직원 수
  - 재직자 수
  - 리더(평가자) 수
  - 부서 수

##### ✍️ 결재자 관리
- 부서/팀별 결재 라인 설정
- 1차 → 2차 → 3차 평가자 지정

## 자주 묻는 질문 (FAQ)

### Q1. 유형효과와 무형효과의 차이는?
**A:**
- **유형효과**: 금액으로 측정 가능한 절감 효과 (예: 불량 감소로 인한 비용 절감)
- **무형효과**: 금액으로 측정하기 어려운 효과 (예: 품질 향상, 안전성 증대)

### Q2. 유형효과 검증은 왜 필요한가요?
**A:** 제안자가 예상한 금액이 실제로 타당한지 검증하기 위함입니다. 검증담당자가 정확한 금액을 확정한 후 평가를 진행합니다.

### Q3. 복수 권한자는 어떤 페이지로 이동하나요?
**A:** 권한 우선순위에 따라 자동 이동합니다.
- admin > effectVerifier > reviewer > user 순서

### Q4. 평가가 지연되면 어떻게 되나요?
**A:**
- 자동 알림 발송 (평가 기한 초과 시)
- 자동 에스컬레이션 (설정된 기한 초과 시 상위자에게 자동 전달)

### Q5. 중복 제안은 어떻게 확인하나요?
**A:**
- 제안 작성 시 "중복 검증" 버튼 클릭
- 평가 시 "중복 제안 검토" 버튼 클릭
- 시스템이 자동으로 유사 제안 검색 (제목, 내용 유사도 분석)

### Q6. 제안서를 수정할 수 있나요?
**A:**
- 제출 전: 수정 가능
- 제출 후: 평가자가 "재검토 요청" 시에만 수정 가능
- 승인/반려 후: 수정 불가
