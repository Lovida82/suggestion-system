## 1. 제안 배경 및 필요성

### 1.1 현재 상황
현재 우리 회사의 제안제도는 그룹웨어 게시판을 통해 운영되고 있으며, 다음과 같은 프로세스로 진행됩니다:

```
[제안 접수] → [수기 정리] → [평가 배정] → [평가 진행] → [등급 산정] → [보상 지급]
  (게시판)    (Excel 등)     (수동 연락)    (별도 문서)      (수기 계산)
```

### 1.2 디지털 전환의 필요성
- 정부 및 산업계의 **디지털 전환(Digital Transformation)** 추세
- 업무 효율화를 통한 **생산성 향상** 요구
- 데이터 기반 의사결정을 위한 **통계 및 분석 체계** 필요
- 직원 편의성 향상을 통한 **제안 참여율 증대** 필요

## 3. 개선안 (제안 내용)

### 3.1 솔루션 개요
**통합 제안제도 관리 시스템**을 직접 개발하여 도입함으로써, 제안 접수부터 평가, 통계까지 전 과정을 **자동화·디지털화**합니다.

### 3.2 주요 기능

#### 3.2.1 사용자 기능 (제안자)
- ✅ **온라인 제안서 작성**: 체계적인 입력 폼 제공
- ✅ **파일 첨부**: 증빙 자료 업로드 (최대 5개, 10MB)
- ✅ **실시간 진행 상황 확인**: 현재 평가 단계 실시간 조회
- ✅ **중복 제안 자동 검색**: AI 기반 유사 제안 자동 검색
- ✅ **알림 수신**: 평가 결과, 반려 사유 등 자동 알림
- ✅ **내 제안 이력 조회**: 과거 제안 통합 관리

#### 3.2.2 평가자 기능
- ✅ **3차 평가 시스템**: 1차 → 2차 → 3차 단계별 평가
- ✅ **자동 배정**: 부서별 평가자 자동 배정
- ✅ **평가 입력 폼**: 노력도, 창의성, 무형효과 체계적 평가
- ✅ **점수 자동 계산**: 종합점수 및 등급 자동 산정
- ✅ **대시보드**: 결재 대기 목록, 처리 현황 실시간 확인
- ✅ **통계 조회**: 개인별 처리 현황, 승인율 등

#### 3.2.3 검증담당자 기능
- ✅ **유형효과 검증**: 금액 절감 효과 체계적 검증
- ✅ **AI 분석 지원**: OpenAI GPT-4 기반 금액 타당성 분석
- ✅ **확정 금액 입력**: 검증된 금액 입력 및 점수 자동 계산
- ✅ **검증 이력 관리**: 과거 검증 내역 조회

#### 3.2.4 관리자 기능
- ✅ **사용자 관리**: 6개 권한 레벨 관리 (일반사용자, 1차/2차/3차 평가자, 검증담당자, 관리자)
- ✅ **조직도 관리**: Excel 업로드를 통한 조직도 자동 반영
- ✅ **결재라인 설정**: 부서별 평가자 지정
- ✅ **통합 대시보드**: 전체 제안 현황, 병목 구간 실시간 모니터링
- ✅ **통계 분석**: 부서별/월별/연간 제안 현황 분석
- ✅ **시스템 설정**: 평가 기준, 등급 기준 관리

### 3.3 기술 구성

| 구성 요소 | 기술 | 특징 |
|-----------|------|------|
| **프론트엔드** | HTML, CSS, JavaScript | 별도 앱 설치 불필요 (웹 브라우저만 필요) |
| **백엔드** | Netlify Functions (Serverless) | 서버 관리 불필요, 자동 확장 |
| **데이터베이스** | Firebase Firestore | 클라우드 기반, 실시간 동기화 |
| **인증** | Firebase Authentication | 보안 인증, 사번 기반 로그인 |
| **AI 분석** | OpenAI GPT-4 API | 유형효과 검증 지원 |
| **호스팅** | Netlify | 99.9% 가용성, CDN 기반 |

```
        ↓

        ↓

└─────────────────────────┘
```
